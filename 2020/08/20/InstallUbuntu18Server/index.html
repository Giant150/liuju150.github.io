在Hyper-V中安装Ubuntu Server 18.04

Posted by 刘巨 on 2020-08-20

新建虚拟机

设置虚拟机名称和安装位置

设置虚机的代数/我的机器只支持第一代，如果机器可以，最好设置为第二代

分配内存，这里设置4GB内存

选择网络

配置虚拟硬盘大小与位置

选择Ubuntu安装包ISO

虚拟机配置汇总

修改CPU处理核心数量

安装Ubuntu

选择语言，这里选择中文

选择安装Ubuntu服务器版

选择系统默认语言，默认为English

因为我这个安装包不是最新的，所以有升级，这里选择默认配置

键盘配置

网络配置，这里用的DHCP自动分配IP地址，也可以配置固定IP地址

设置代理服务器地址，没有默认为空

这里设置镜像服务器位置，本机修改为华为云的镜像站
https://repo.huaweicloud.com/ubuntu/

文件系统配置，都选择默认

选择硬盘

硬盘信息汇总

确认格式化硬盘

配置用户名/密码

SSH配置，是否安装SSH Server

选配安装组件

等待安装
